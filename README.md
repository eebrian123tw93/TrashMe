## TrashMe

<img src="./screenshots/trash_me.png" width="100%"/>

>以到府收取垃圾為服務的APP。
### 特色
* Server端使用Kotlin + Spring Boot
* App端為Android，RestfulAPI + Web Socket連線讓用戶可及時查看服務員當前位置
* MVP架構
